# Code of Conduct

**USF BIOS - US Inc Training & Fine-tuning Platform**

---

## Proprietary Software

This software is the exclusive property of **US Inc**. All users must comply with the terms of their license agreement.

## Professional Conduct

All authorized users of USF BIOS are expected to:

1. **Maintain Confidentiality**: Do not share, distribute, or disclose any proprietary information, source code, algorithms, or documentation without explicit written permission from US Inc.

2. **Respect Intellectual Property**: Acknowledge that all intellectual property rights belong exclusively to US Inc.

3. **Authorized Use Only**: Use the software only for purposes explicitly permitted under your license agreement.

4. **Security**: Maintain the security of your access credentials and report any security incidents immediately.

5. **Compliance**: Adhere to all applicable laws and regulations when using this software.

## Prohibited Actions

The following actions are strictly prohibited:

- Unauthorized copying, modification, or distribution of the software
- Reverse engineering, decompiling, or disassembling the software
- Sharing access credentials with unauthorized parties
- Using the software for purposes outside the scope of your license
- Attempting to circumvent any security measures

## Reporting

Report any violations or concerns to:

- **Email**: compliance@us.inc
- **Security Issues**: security@us.inc

## Enforcement

Violations of this Code of Conduct may result in:

- Immediate termination of license
- Legal action
- Liability for damages
